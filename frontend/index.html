AutoDRIVE Race Control Tower (RCT) is a WebSocket race-control proxy for the AutoDRIVE RoboRacer simulator. It accepts simulator traffic, forwards vehicle data to configured DevKit bridge instances, rewrites vehicle identifiers for each assigned RoboRacer, and streams connection status back to this browser dashboard.
